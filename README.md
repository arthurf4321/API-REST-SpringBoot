# Clínica Beatriz - API (Teste)

Esse projeto é uma API REST simples que estou desenvolvendo como parte inicial de um sistema para clínica.

No momento, o foco é entender e testar o funcionamento de um backend com Spring Boot, incluindo conexão com banco de dados e operações básicas com clientes.

## Tecnologias

- Java
- Spring Boot
- MySQL

## Objetivo

Esse projeto vai ser feito com o objetivo de aprendizagem e fixicao de conteudo Spring Boot para aplicar nas versoes futuras do projeto clinica_beatriz.
